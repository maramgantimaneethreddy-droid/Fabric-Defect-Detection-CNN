## Project Overview

This project focuses on automatic fabric defect detection using deep learning.

The objective is to classify fabric images into six categories using a custom Convolutional Neural Network (CNN) and compare its performance with MobileNetV2 transfer learning.

## Dataset

Fabric Defects Dataset

https://www.kaggle.com/datasets/nexuswho/fabric-defects-dataset
